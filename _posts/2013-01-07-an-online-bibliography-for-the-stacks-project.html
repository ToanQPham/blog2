Ever found yourself browsing the Stacks project and encountering a citation? I did, and I found it quite annoying that I couldn't get any information, unless I opened the bibliography file and started looking for the reference. Not really user-friendly... So I decided to implement an online bibliography :).

The bibliography overview page contains all items from the bibliography, and whenever a work is cited in the text, see for instance tag 03B3 you get a link to the page of that particular item.

At some point this will be improved, parsing arXiv links, making things more consistent, giving overviews per author (maybe), etc. But for now everything is quite workable I hope.
