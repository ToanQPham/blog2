Every once in a while I want to compute the Hochschild cohomology of a finite-dimensional algebra which is given as a quiver with relations. Luckily, provided the quiver isn't too big, this can be done quite easily using QPA which is a package for GAP.

As an example to see how this is done, let us consider the Beilinson quiver for $\mathbb{P}^2$, which is

and the relations are $x_0y_1-y_0x_1, x_0z_1-z_0x_1, y_0z_1-z_0y_1$. Then there exists a derived equivalence $\mathbf{D}^{\mathrm{b}}(\mathbb{P}^2)\cong\mathbf{D}^{\mathrm{b}}(kQ/I)$. There are various definitions of the Hochschild cohomology of $\mathbb{P}^2$, all of them agree (luckily), and Hochschild cohomology being a derived invariant we might as well compute the Hochschild cohomology of $\mathbb{P}^2$ as the Hochschild cohomology of this finite-dimensional algebra. But if we denote $A=kQ/I$, this is nothing but $\mathrm{Ext}_{A\otimes_kA^{\mathrm{op}}}^n(A,A)$. And computing this Ext can be done in QPA. Without further ado, here is the code. {% highlight gap %} LoadPackage("qpa");; Q := Quiver(["p", "q", "r"], [ ["p", "q", "x0"], ["p", "q", "y0"], ["p", "q", "z0"], ["q", "r", "x1"], ["q", "r", "y1"], ["q", "r", "z1"] ]);; kQ := PathAlgebra(Rationals, Q);; AssignGeneratorVariables(kQ);; relations := [x0*y1-y0*x1, x0*z1-z0*x1, y0*z1-z0*y1];; ExtAlgebraGenerators(AlgebraAsModuleOverEnvelopingAlgebra(kQ / relations), 4); {% endhighlight %}

So all we need to know is that there exists something like ExtAlgebraGenerators to compute the algebra structure of the Ext-algebra, and a way to get $kQ/I$ as a module over its enveloping algebra using AlgebraAsModuleOverEnvelopingAlgebra. Easy peasy lemon squeezy.

From this we read off that $\operatorname{dim}_k\operatorname{HH}^i(kQ/I)$ is 1, 8 and 10 for $i=0,1,2$. We can also read off that as an algebra it is generated by its piece in degree 1. Then a vector space basis for this algebra is given.

Of course, for $\mathbb{P}^2$ we could also have used the Hochschild–Kostant–Rosenberg decomposition, expressing the Hochschild cohomology in terms of polyvector fields. But the current method allows us to compute the Hochschild cohomology of noncommutative planes (albeit case by case) by changing the relations in the quiver accordingly. I have written up a description of these Hochschild cohomologies in all cases of the classification using geometric techniques, which will appear on the arXiv shortly.
